# Source Dataset

This study uses open DEM tiles from AWS Terrain Tiles:

- SRTM-derived GeoTIFF tiles at zoom 12

Terrain families:

- flat floodplain
- moderate river valley
- basin storage terrain
- steep terrain edge
- mixed relief terrain

Every extent is small, reproducible, and open-data-derived.

The study uses one fixed coefficient first: `1e-6`.
