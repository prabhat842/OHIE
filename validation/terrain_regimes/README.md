# Terrain Regimes Study

This benchmark tests whether the flux-coupling coefficient behaves differently across terrain regimes.

Scope:

- 5 open-DEM micro-extents only
- one fixed coefficient first: `1e-6`
- no tuning toward success

Purpose:

- characterize where transfer is strong, moderate, or weak
- derive a reviewer-defensible terrain-regime hypothesis

This is a transfer characterization study, not calibration and not solver development.
