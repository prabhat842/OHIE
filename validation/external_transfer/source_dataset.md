# Source Dataset

This benchmark uses one open DEM tile from AWS Terrain Tiles:

- `ohie-data/external_transfer/brahmaputra_dhakuakhana_z12.tif`

Selection rationale:

- Brahmaputra floodplain micro-extent
- Dhakuakhana area on the north-bank floodplain
- simple river-adjacent terrain
- open DEM only
- 5-10 km study window via a single z12 terrain tile

Scientific status:

- open-data transfer test
- not a calibrated validation dataset
- no claim of operational accuracy
- no claim of river-geometry fidelity beyond the simple edge forcing used here
