# External Transfer Benchmark

This benchmark runs one open-DEM transfer case only.

Selection:

- Brahmaputra floodplain micro-extent near Dhakuakhana, Assam
- AWS Terrain Tiles GeoTIFF z12 tile
- open DEM only
- simple river-adjacent edge forcing

Purpose:

- test whether the existing flux-coupling approximation transfers outside the original benchmark family
- document failure or partial transfer honestly

This is a transferability test, not a validation claim and not a Brahmaputra flood model.
