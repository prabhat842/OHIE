# Rann / Gadkabet Approximate Benchmark

Purpose: test OHIE behavior on flat terrain with ponding, blue spots, drainage spines, stagnation, and road-ridge damming behavior.

This is **not calibrated** and does not use real Gadkabet DEM/SAR/tidal creek data.

Run:

```bash
python3 validation/historical/rann_gadkabet/run.py
```
