# Flat Terrain Real-Data-Informed Benchmark

This case runs OHIE on a small local historical-project-derived terrain chip.

Purpose:

- test low-gradient routing behavior
- measure persistence and stagnation proxies
- compare D8 and D-Infinity routing diagnostics
- keep the experiment fully rerunnable from repository-local data

Scientific status:

- reproducible software benchmark
- not calibrated
- not an operational flood map
