# Expected Behavior

- Water should accumulate in lower portions of the terrain chip.
- Persistence should be nonzero where shallow ponding remains.
- D-Infinity should allow at least some split routing where the local slope direction is not aligned to a single D8 receiver.
- Mass error should remain small enough for a smoke validation case.

This case is expected to show plausible low-gradient behavior, not calibrated real-world flood accuracy.
