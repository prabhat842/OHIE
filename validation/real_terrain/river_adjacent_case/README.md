# River-Adjacent Real-Data-Informed Benchmark

This case runs OHIE on a small terrain chip with a synthetic river-edge mask.

Purpose:

- test whether prescribed river stage changes model state
- verify boundary forcing plumbing
- compare rainfall-only and staged-boundary outcomes

Scientific status:

- reproducible boundary-response benchmark
- river mask is synthetic
- not calibrated to a measured river hydrograph
- not full river-urban drainage coupling
