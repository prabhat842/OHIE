# Phase 6 Remote Sensing Comparison

This folder contains the first OHIE observation-mask comparison scaffold.

Current status:

> **proxy comparison only**

The packaged mask comes from a local historical flood-output raster. It is not raw Sentinel-1 SAR, NDWI, Copernicus EMS, or JRC water data.

The metrics are useful for validating the comparison workflow:

- IoU
- observed overlap percent
- flooded area agreement

They should not be interpreted as publication-grade remote-sensing validation yet.
