# Known Limitations

OHIE is intentionally limited.

## Scientific limitations

- empirical coefficient, not calibrated
- simplified hydraulics, not full coupled river-urban flow
- terrain dependence is real
- mixed-relief transfer weakens
- external transfer evidence is limited to one case
- no operational forecasting claim

## Practical limitations

- large raster datasets are not bundled
- external open-data preprocessing is still minimal
- benchmark outputs are for reproduction and critique, not design-grade use

## Interpretation rule

If a claim is not supported by `docs/scientific_position.md`, it should not be treated as a project claim.
