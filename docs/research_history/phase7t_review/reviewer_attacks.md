# Phase 7T Reviewer Attacks

## Skeptical Hydrologist

Strongest remaining criticism:

- The coefficient is explainable, but the transfer sweep shows it is still an empirical benchmark-scale conductance rather than a transferable river-plain boundary law.

Confidence increase:

- The sweep has made the approximation class explicit and has identified a failure boundary instead of hiding it.

Unresolved concern:

- No field calibration, no measured river hydrograph, and no shared stable region across terrain families.

Next scientific bottleneck:

- either a terrain-aware coefficient law or an external open-data transfer case with documented behavior.

## Flood Modeler

Strongest remaining criticism:

- The model still uses a simplified diffusive-wave core plus exchange approximation, not full hydraulic coupling.

Confidence increase:

- The transferability study shows where the approximation fails, which is scientifically more useful than silent local tuning.

Unresolved concern:

- Response remains approximate and should not be described as transferable coupled hydraulics.

Next scientific bottleneck:

- test whether a modified boundary law can recover a shared stable region without hiding mass-error growth.

## Municipal Engineer

Strongest remaining criticism:

- The model still does not rank interventions or solve operational design tradeoffs.

Confidence increase:

- The boundary is less likely to be mistaken for a universal default.

Unresolved concern:

- Still not design-grade infrastructure sizing or operations support.

Next scientific bottleneck:

- keep the language frozen while improving benchmark breadth.

## Reproducibility Reviewer

Strongest remaining criticism:

- The study is reproducible, but the terrains are still benchmark families rather than external open-data-native basins.

Confidence increase:

- The coefficient sweep and failure modes are explicit across multiple terrain classes.

Unresolved concern:

- No external terrain transfer dataset yet, and no common stable region emerged from the internal sweep.

Next scientific bottleneck:

- add a single external open-data transfer case only if it can be documented cleanly and compared against the current failure pattern.
