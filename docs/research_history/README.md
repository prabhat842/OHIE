# Research History Archive

This folder preserves the internal scientific trail that led to the public OHIE position.

It includes:

- phase review folders
- phase reports
- internal documentation that is useful for traceability but too detailed for the public root

The public repository should be read from:

- `README.md`
- `docs/scientific_position.md`
- `docs/scientific_summary.md`
- `docs/benchmark_guide.md`
- `docs/evidence_matrix.md`

The archive exists for reproducibility, publication support, and methodology traceability.
