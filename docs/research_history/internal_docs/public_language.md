# Public Language Freeze

Use the following language for OHIE Phase 7R and earlier unless stronger evidence exists.

## Allowed Terms

- compound forcing scaffold
- intervention comparison framework
- benchmark reproduction
- real-terrain sanity testing
- scientifically serious scaffold
- flux-coupled boundary approximation
- proxy comparison only

## Forbidden Terms

Do not use the following until the evidence base changes:

- calibrated urban flooding
- intervention intelligence
- validated flood forecasting
- coupled river-urban hydraulics
- operational flood resilience engine

## Rules

- Do not elevate proxy comparisons into validation.
- Do not elevate scenario comparison into optimization.
- Do not elevate a flux approximation into coupled hydraulics.
- Do not use “validated” unless the benchmark really justifies it.

## Safe sentence patterns

- “OHIE provides a benchmark reproduction scaffold.”
- “OHIE includes a flux-coupled boundary approximation.”
- “OHIE supports intervention comparison.”
- “OHIE is a scientifically serious scaffold, not a calibrated operational model.”
