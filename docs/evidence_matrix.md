## Reading guide

- Use Phase 7 for the boundary-defensibility change.
- Use Phase 7T for the failed shared-default claim.
- Use Phase 7X for the one external open-DEM transfer signal.
- Use Phase 7Y for the terrain-regime pattern.

The matrix is intentionally conservative. It maps claims to evidence, not evidence to marketing language.
