# Scientific Position

OHIE is a bounded empirical hydrodynamic research scaffold.

It exists to support intervention-aware experimentation, a flux-coupled boundary approximation, and a reproducible benchmark framework.

## What OHIE Is

- a benchmark reproduction scaffold for controlled hydrodynamic experiments
- an intervention comparison framework for testing scenario differences
- a flux-coupled boundary approximation that is more defensible than overwrite forcing
- a scientifically serious scaffold for studying where the approximation behaves credibly

## What OHIE Is Not

OHIE is not:

- a calibrated hydrodynamic engine
- an operational forecasting system
- a design-grade drainage model
- validated intervention intelligence
- a coupled river-urban hydraulic model

## Current Confidence

| Capability | Confidence |
|---|---|
| Flat terrain routing | Medium |
| Floodplain transfer | Medium |
| External open-DEM transfer | Medium-Low |
| Mixed-relief transfer | Low |
| Flux-coupled boundary approximation | Low-Medium |
| Hydraulic infrastructure realism | Low |
| Calibration | Low |
| Real-terrain sanity testing | Medium |

## Known Weaknesses

- terrain dependence is real
- the coefficient is empirical, not calibrated
- mixed-relief terrain weakens transfer
- external transfer evidence is limited to one case
- hydraulics are simplified rather than fully coupled

## Current Scientific Hypothesis

The current conductance approximation appears to transfer most credibly in low-gradient and basin-like terrain and weakens as relief heterogeneity increases.

This is a hypothesis only.

It is not a law.
It is not calibration.
It is not universal transferability.
