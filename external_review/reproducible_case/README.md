# Reproducible Case

This folder points reviewers to the one-case external transfer benchmark.

Run:

```bash
python3 validation/external_transfer/run.py
```

Inspect:

- `validation/external_transfer/observed_output.md`
- `ohie-phase7x-external-transfer-report.md`

The intent is to let external reviewers reproduce a single transfer case without searching the repository for entry points.
