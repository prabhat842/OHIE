# Quickstart

This package is for external review, not for feature development.

## Run the reproducible benchmark

```bash
python3 validation/external_transfer/run.py
```

## What to inspect

- `../validation/external_transfer/observed_output.md`
- `../ohie-phase7x-external-transfer-report.md`
- `../docs/scientific_position.md`
- `../docs/evidence_matrix.md`

## What the reviewer should look for

- whether the external open-DEM case behaves like a bounded empirical approximation
- whether the result still depends on terrain regime
- whether the claims stay inside the evidence

This is a single-case transfer test.

It is not calibration and not validation.
